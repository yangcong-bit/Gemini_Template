# 第十五届省赛 核心数据字典建模 (SystemData_t)

| 数据含义 (UI展示或逻辑运算) | 对应 sys 字典变量名 | 数据类型 | 默认值/作用域 | 备注说明 (题目要求) |
| :--- | :--- | :--- | :--- | :--- |
| **【底层采集原始量】** | | | | |
| 通道 A 原始脉冲频率 | `freq_ch1` | `uint32_t` | 0 (只读) | 由底层 `freq_app.c` 自动更新 |
| 通道 B 原始脉冲频率 | `freq_ch2` | `uint32_t` | 0 (只读) | 由底层 `freq_app.c` 自动更新 |
| **【核心换算物理量】** | | | | |
| 通道 A 校准后频率 | `f_a_cal` | `int32_t` | 0 (全局) | 公式: `freq_ch1 + PX` (可能为负数) |
| 通道 B 校准后频率 | `f_b_cal` | `int32_t` | 0 (全局) | 公式: `freq_ch2 + PX` (可能为负数) |
| **【系统设置参数】** | | | | |
| 频率突变参数 (Delta) | `para_pd` | `uint32_t` | 1000 | 步进100, 范围: 100Hz ~ 1000Hz |
| 频率超限参数 (High) | `para_ph` | `uint32_t` | 5000 | 步进100, 范围: 1000Hz ~ 10000Hz |
| 频率校准参数 (X) | `para_px` | `int32_t` | 0 | 步进100, 范围: -1000Hz ~ 1000Hz |
| **【统计记录数据】** | | | | |
| A通道频率突变次数 | `count_nda` | `uint32_t` | 0 | 3秒滑动窗口最大差值 > PD 时加1 |
| B通道频率突变次数 | `count_ndb` | `uint32_t` | 0 | 同上 |
| A通道频率超限次数 | `count_nha` | `uint32_t` | 0 | `f_a_cal > PH` (边缘触发加1) |
| B通道频率超限次数 | `count_nhb` | `uint32_t` | 0 | 同上 |
| **【UI 交互与状态】** | | | | |
| 频率/周期显示模式 | `data_mode` | `uint8_t` | 0 (全局) | 0:频率(Hz), 1:周期(uS)。B3 切换 |
| 参数焦点选择 | `para_focus`| `uint8_t` | 0 (局部) | 0:PD, 1:PH, 2:PX。放 `Logic_Ctrl_Proc` 做 `static` 即可 |
